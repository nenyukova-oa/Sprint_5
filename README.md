# Sprint_5
Автотесты для сервиса Stellar Burgers

Файлы в папке с тестами:

1. test_registration: тесты на регистрацию пользователя:
- успешная регистрация
- ошибка для некорректного пароля
2. test_entry_exit: тесты на вход и выход из аккаунта зарегистрированного пользователя:
- вход по кнопке «Войти в аккаунт» на главной
- вход через кнопку «Личный кабинет»
- вход через кнопку в форме регистрации
- вход через кнопку в форме восстановления пароля
- выход по кнопке «Выйти» в личном кабинете
3. test_transitions: тесты на переход по сайту:
- переход по клику на «Личный кабинет»
- переход по клику на «Конструктор» из личного кабинета
- переход по клику на логотип Stellar Burgers из личного кабинета
- работает переход к разделу "Начинки"
- работает переход к разделу "Соусы"
- работает переход к разделу "Булки"

Файлы в проекте:
1. locators: локаторы
2. curl: ссылки для тестирования сайта
3. data: данные для регистрации и входа
4. conftest: фикстуры
